Portfólio

Perfil Game

Site para gerar cartas online do clássico jogo de adivinhação “Perfil”, onde jogadores devem descobrir uma palavra secreta a partir de 20 dicas. Criado com o propósito de aplicar conceitos de lógica de programação, JavaScript e JSON.

Provérbios Bíblicos

Site que exibe versículos do livro bíblico de Provérbios, com funcionalidades para visualização aleatória ou busca por palavra-chave. Com uma interface leve e responsiva, o projeto foi pensado para proporcionar uma experiência amigável em diferentes dispositivos, sem o uso de frameworks ou bibliotecas externas — apenas HTML, CSS, JavaScript e JSON.
